## 段落格式

这是一级标题下面的文字，末尾添加两个空格以上表示换行

或者使用空行来换行。

*斜体文字*- --> 一个星号包围

**粗体文字** --> 两个星号包围

三个以上的空行星号表示分割线

****

~~删除线~~ --> 两个波浪线表示删除线

<u>下划线</u> --> < u > 包围

脚注[^要注解的文字] 加上[^]内容
